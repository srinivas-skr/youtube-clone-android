# youtube-clone-android

An Android app project that demonstrates a YouTube-style client, built with industry-standard tools and structure. This project is suitable for learning, portfolio, and HR review.

## Project Highlights
- **Modern Android app** using Gradle, Kotlin/Java, and recommended architecture.
- **Clean codebase**: Activities, Fragments, ViewModels, and modular resources.
- **Media playback**: Integrates with supported Android media APIs.
- **Easy to build and run**: Works in Android Studio or with command-line Gradle.

## How to Build & Run
1. Open in Android Studio (File → Open → select this folder), or use terminal:
	```bash
	./gradlew assembleDebug
	```
2. Install APK on device/emulator:
	```bash
	adb install -r app/build/outputs/apk/debug/app-debug.apk
	```

## Folder Structure
- `app/src/main/AndroidManifest.xml` — app configuration and permissions
- `app/src/main/java` or `kotlin` — main source code
- `app/src/main/res/` — layouts, images, strings, themes
- `build.gradle.kts` — project build settings
